# Brain-Tumor-Classification
Compared 5 CNN architectures (ResNet50, EfficientNetB0,  DenseNet121, MobileNetV2, InceptionV3) to classify 3 types of brain tumors from MRI scans.  DenseNet121 achieved the best performance with 97% accuracy and 96% macro F1-score.  Tools: TensorFlow, Keras, OpenCV, Transfer Learning, Fine-tuning, Data Augmentation, Image  Processing
